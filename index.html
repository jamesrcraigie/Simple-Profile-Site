Masters of Dugeons

Master of Dugeons is a 3D survival horror game build in C# with Unity. It comprises of two levels. The first level players solve a puzzle and escape a maze while trying to avoid zombies. The faster the time, the higher the score.

The second level, the player must try survive in a dungeon style map and avoid and kill as many zombies as possible. The more zombies they killed before dying, the higher they scored.

The maze uses a concpet known as Procedural Content Generation (PCG), which means they are created using algorithms, so the player will face a new maze with each play. Within the maze the player has to find coloured digits which together provided the key to escape.

In the main menu, players had the choice of algorithms for the maze generation. Each algorithm provided a slightly varied dynamic to the maze. Most of the algorithms were established, known algorithms for generating maze, but i did write a genetic algorithm for content generation. Genetic algorithms are based on using the Darwinian concept of evolution to find an optimal solution to a given problem. Players could also view a global leaderboard, linked to an API, to see how they faired against players worldwide.
